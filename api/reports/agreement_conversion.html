Reports API changes after conversion to new services

Bring is revising the service portfolio. Some of our services have been given new service names, service codes and pricing models. Read more about the changes

Old services will be phased out. From a given date, your company will be converted to use revised services during Booking, and you can no longer book old services. All customers belongs to your company will be converted. The revised services are booked using main customer number.

In Reports API , we have made changes to support Main Customer Number for some service types and for some service types new definitions are available.
API users after conversion need to use appropriate customer number and definition combination to get the correct result set from the reports.

We recommend API users to execute List available customers and List available reports for a customer to understand what parameters to be used for report execution.
